INTRODUCTION
------------

The "Remove Search Result Title" Module is a tiny Drupal 8 Module to remove
the default search result title. 

It originally based on 
<https://drupal.stackexchange.com/questions/198010/change-markup-shown-before-the-search-result>


REQUIREMENTS
------------

No requirements.

RECOMMENDED MODULES
-------------------

No recommended modules.

INSTALLATION
------------

This module is currently not listed at [drupal.org](https://www.drupal.org/). I suggest an installation into the `modules/custom` directory using git clone:

```bash
git clone https://gitlab.com/meengit/rmsearchrestitle.git
```
